Результат модуля: уверенно ориентируетесь в компьютере и среде разработчика, не теряетесь в интерфейсах и понимаете, где происходит магия.

03 Модуль 2. GitHub и мышление разработчика

Результат модуля: умеете хранить и показывать свои проекты, публиковать их в GitHub.

Уроки:

  • Урок 1. Что такое GitHub и зачем он
  • Урок 2. Работа с репозиториями (создание, загрузка файлов, коммиты)
  • Урок 3. Практика: загрузка проекта (HTML), публичная ссылка
04 Модуль 3. Первый код и первый результат

Результат модуля: знаете, что код — это просто инструкции; создали свою первую веб‑страницу.

Уроки:

  • Урок 1. Что такое HTML (логика, структура, первая страница в Cursor)
  • Урок 2. CSS — красота и стиль (цвета, шрифты, отступы, практика)
  • Урок 3. Мини‑проект (личная страница / визитка, ИИ, ошибки и безопасность)
05 Модуль 4. Вайб‑кодинг и no‑code (lovable.dev)

Результат модуля: создаёте сервисы без классического кода. Готовый онлайн‑сервис по ссылке.

Уроки:

  • Урок 1. Знакомство с lovable.dev (регистрация, интерфейс, примеры)
  • Урок 2. Создание сервиса (формы, кнопки, поля, логика, связка с ИИ)
  • Урок 3. Публикация и тестирование (публикация, проверка на устройствах, GitHub, домен)
06 Только тариф PRO Модуль 5. Аналоги сервисов для вайб‑кодинга

Уроки:

  • Урок 1. Аналоги Cursor
  • Урок 2. Аналоги lovable
07 Практика: Итоговый проект

Результат: закрепили навыки создания, улучшения и оформления; создали полноценный финальный проект (идея, формат, реализация с ИИ, публикация в GitHub, демонстрация, обратная связь).
